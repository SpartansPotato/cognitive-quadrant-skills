# Changelog

## 1.1.0 — 2026-07-17

### Added

- Cross-platform support for WorkBuddy, OpenAI Codex, and Claude Code.
- Platform-neutral Skill instructions with language matching.
- English-first trigger descriptions with Simplified Chinese trigger phrases.
- Codex `agents/openai.yaml` UI metadata.
- English and Chinese platform installation guides.
- macOS/Linux and Windows installation scripts for Codex and Claude Code.
- Structural validation script and GitHub Actions workflow.
- English reference documents and complete editable English Word template packs matching the Chinese editions.
- Universal release ZIPs usable across the three supported hosts.

### Changed

- Renamed Skill folders to platform-neutral names without the `-cn` suffix.
- Reduced the main `SKILL.md` files and moved detailed guidance into references for progressive disclosure.
- Replaced WorkBuddy-only assumptions with capability detection and explicit fallback behavior.
- Updated repository documentation to English-first bilingual presentation.

### Preserved

- Chinese WorkBuddy manifests and complete editable Chinese Word template packs.
- Domestic-model and restricted-network adaptations.
- General cognition and teacher-research specializations.
