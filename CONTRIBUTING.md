# Contributing

Thank you for improving Cognitive Quadrant Skills.

## Good contributions

- clearer trigger descriptions;
- new bilingual examples;
- evidence-quality and safety improvements;
- WorkBuddy, Codex, or Claude Code compatibility fixes;
- reproducible evaluation cases;
- accessible templates for non-technical users;
- corrections to theory attribution or platform documentation.

## Before opening a pull request

1. Run `python scripts/validate_skills.py`.
2. Keep `SKILL.md` concise and move detailed material into `references/`.
3. Update English and Chinese documentation together when behavior changes.
4. Do not add secrets, API keys, copyrighted database exports, personal student data, or unlicensed templates.
5. Do not claim that exploratory hypotheses are established facts.

## Issues

A good issue includes:

- host platform and version;
- model used, when relevant;
- installation scope;
- exact prompt;
- expected behavior;
- observed behavior;
- screenshots or logs with private information removed.
