# Cognitive Quadrant Skills v1.1.0

## English

Version 1.1.0 is the first cross-platform release. The same two core Skills are now packaged for WorkBuddy, OpenAI Codex, and Claude Code.

### Highlights

- platform-neutral core workflow;
- English-first bilingual activation descriptions;
- user-language matching;
- Codex UI metadata;
- WorkBuddy manifest compatibility;
- Codex and Claude Code install scripts;
- bilingual installation guides and references;
- complete editable English and Simplified Chinese Word template packs;
- automated structural validation;
- universal ZIP packages for all three hosts.

### Downloads

- `cognitive-quadrant-general-v1.1.0.zip`
- `cognitive-quadrant-teacher-research-v1.1.0.zip`

### Important note

The packages follow each platform’s documented Skill conventions, but available web, file, document, and execution tools still depend on the host, selected model, and permissions.

## 中文

v1.1.0 是首个跨平台版本。同一套两个核心 Skill 现在可以用于 WorkBuddy、OpenAI Codex 和 Claude Code。

### 主要更新

- 平台中立核心工作流；
- 英文优先、中文补充的双语触发描述；
- 自动跟随用户语言；
- Codex 界面配置；
- 保留 WorkBuddy `manifest.yaml`；
- Codex 与 Claude Code 自动安装脚本；
- 双语安装说明和参考规则；
- 完整的英文版与简体中文版可编辑 Word 模板包；
- 自动结构校验；
- 三个平台共用的通用 ZIP 安装包。

### 下载文件

- `cognitive-quadrant-general-v1.1.0.zip`
- `cognitive-quadrant-teacher-research-v1.1.0.zip`

### 重要说明

工具包已经按照三个平台公开文档的 Skill 结构进行适配，但实际可用的联网、文件、文档生成和执行工具仍取决于宿主平台、模型和权限。
