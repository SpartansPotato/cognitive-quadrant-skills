# Install in WorkBuddy

1. Download one of the universal ZIP packages from `releases/`.
2. Open WorkBuddy Skill management.
3. Choose the option to upload/import a local Skill.
4. Select the ZIP and enable the imported Skill.
5. Test with a prompt from the Skill README.

The package includes the WorkBuddy-required `SKILL.md` and `manifest.yaml`, plus references and assets. It does not require an external API key by default.

Menu wording can vary by WorkBuddy edition and enterprise policy. If local Skill upload is disabled, ask the administrator to upload and distribute the package.
