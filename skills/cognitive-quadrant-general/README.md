# Cognitive Quadrant — Thinking & Exploration

Cross-platform Skill for WorkBuddy, Codex, and Claude Code. It helps users clarify issues, identify knowledge gaps and blind spots, revise assumptions, explore new possibilities, and design testable next steps.

- Codex explicit invocation: `$cognitive-quadrant-general`
- Claude Code explicit invocation: `/cognitive-quadrant-general`
- WorkBuddy: import the universal ZIP or folder package.

The Skill answers in the user's language.


## Bilingual editable templates

Complete editable Word templates are included in both English and Simplified Chinese under `assets/templates/en/` and `assets/templates/zh-CN/`. The two language packs use matching structures so the host can generate reports in the user's language.
