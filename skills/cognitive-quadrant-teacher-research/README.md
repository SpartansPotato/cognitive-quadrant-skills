# Cognitive Quadrant — Teacher Research & Innovation

Cross-platform Skill for educators using WorkBuddy, Codex, or Claude Code. It converts teaching practice into researchable problems, maps evidence gaps, explores theory, revises assumptions, and develops innovation directions with verification paths.

- Codex explicit invocation: `$cognitive-quadrant-teacher-research`
- Claude Code explicit invocation: `/cognitive-quadrant-teacher-research`
- WorkBuddy: import the universal ZIP or folder package.

The Skill answers in the user's language.


## Bilingual editable templates

Complete editable Word templates are included in both English and Simplified Chinese under `assets/templates/en/` and `assets/templates/zh-CN/`. The two language packs use matching structures so the host can generate reports in the user's language.
