# Deliverables

Create files only when the user requests them or when a substantial task clearly benefits from a saved artifact and the host has the required capability.

Possible outputs:

- editable Word or Markdown report;
- quadrant summary image;
- spreadsheet evidence or decision table;
- learning roadmap;
- research diagnosis and innovation report.

Before claiming completion, verify the file exists and provide the actual path or attachment supported by the host. If file generation is unavailable, provide a complete copy-ready structure and say that no file was created.

Images should contain only the most important 3–5 items per quadrant. Do not compress a full report into a crowded graphic.
